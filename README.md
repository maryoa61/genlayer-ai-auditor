# GenLayer AI Auditor

A decentralized security auditing tool built specifically for the GenLayer ecosystem. This project leverages LLMs to provide real-time security analysis for Intelligent Contracts, helping to prevent honeypots, rug-pulls, and common vulnerabilities before deployment.

## Features
- **AI-driven Vulnerability Detection**: Deep analysis of Solidity code targeting reentrancy, ownership abuse, and unauthorized minting.
- **On-chain Ready**: Outputs standardized, machine-readable JSON for integration with GenLayer's "Optimistic Democracy" consensus.
- **Optimized for Speed**: Uses GPT-4o-mini to provide fast, cost-effective security assessments.
